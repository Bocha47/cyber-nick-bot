## 🎯 О проекте

**CyberNick AI** — Telegram бот для генерации уникальных ников и паролей с использованием нейросетей.

### ✨ Возможности

- 🎮 **4 стиля генерации** (Крутой, Аниме, Фэнтези, AI)
- 🧠 **Нейросетевая генерация** через OdiRouter API (free-gpt-5.5)
- 🎨 **Создание аватарок** через Stable Diffusion (HuggingFace)
- 🔐 **Криптостойкие пароли** (16 символов)
- 📜 **История генераций**
- 🌍 **Мультиязычность** (Русский / English)
- ⭐ **3 бесплатные генерации в день**
- 💰 **Монетизация через Telegram Stars**
